Resumen

1. Punto de Acceso Rogue (AP): Un punto de acceso inalámbrico no autorizado dentro de una red, utilizado por atacantes para interceptar, espiar o manipular datos.

2. Ataque Man-in-the-Middle (MitM): Un tipo de ataque en el que el atacante intercepta y retransmite en secreto la comunicación entre dos partes, con la capacidad de alterar la información.

3. Suplantación de SSID: El acto de imitar el nombre de una red inalámbrica legítima para engañar a los usuarios y hacer que se conecten a un punto de acceso rogue.

4. Sistema de Detección de Intrusiones en la Red (NIDS): Un sistema diseñado para detectar accesos no autorizados o actividades maliciosas dentro de una red.

5. Filtrado de Direcciones MAC: Una medida de seguridad que restringe el acceso a la red en función de las direcciones de hardware específicas de los dispositivos.
